# Jigyasa News

A GitHub-ready static front-end rebuilt from the shared Gemini concept page.

## Files

- `index.html` - app structure
- `style.css` - styles and responsive layout
- `script.js` - theme toggle, character counter, starter button logic

## Use

1. Upload these files to a GitHub repository.
2. Open `index.html` locally, or deploy with GitHub Pages.
3. Replace the alert inside `script.js` with your real API call.
